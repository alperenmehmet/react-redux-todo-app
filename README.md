[https://react-redux-todo-3app.netlify.app/](https://react-redux-todo-3app.netlify.app/)

### Clone, Install and Start

```
git clone https://github.com/alperenmehmet/react-redux-todo-app.git
cd react-redux-todo-app
yarn install
yarn start
```
